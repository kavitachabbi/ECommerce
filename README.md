# 🌿 EcoLoop - Circular Economy Ecosystem

EcoLoop is a modular, high-performance circular economy ecosystem built as a "Clean-Tech" React application. It integrates sustainable shopping, peer-to-peer renting, food rescue, and upcycling into a single unified platform.

## 🚀 Key Features
- **The Green Marketplace**: E-commerce with real-time Carbon Footprint tracking and Carbon Offset donations.
- **Neighborhood Swap/Rent**: Interactive SVG Map with verified local listings.
- **Food Rescue Portal**: Real-time surplus feed connecting restaurants with shelters and volunteers.
- **Upcycling Hub**: Connecting waste donors with creative creators.
- **Gamified Dashboard**: Bento-grid layout with circular ranking progress and reward systems.

## 🛠️ Tech Stack
- **Framework**: React 18 (Vite)
- **Styling**: Tailwind CSS (Custom Sage Green palette + Glassmorphism)
- **Navigation**: React Router 6
- **Animation**: Framer Motion (Page transitions, list staggering, haptic feedback)
- **Icons**: Lucide-React
- **State**: React Context API

## 📦 Installation & Setup

1. **Clone the project** and navigate into the directory.
2. **Install all dependencies**:
```bash
npm install framer-motion lucide-react react-router-dom clsx tailwind-merge
```
3. **Configure Tailwind**: Ensure your `tailwind.config.js` includes the custom Sage Green theme and `rounded-3xl` utilities as provided in the source code.
4. **Run development server**:
```bash
npm run dev
```
